               HR Management System
                 Project Over view



1. Project Introduction
   The HR Management System is a web-based application developed using PHP, MySQL, HTML, CSS, and JavaScript. It helps organizations manage employee records, attendance,   payroll, leave requests, and performance evaluations in a centralized and secure environment.


2. Project Objectives
   To digitalize employee management operations.
   To provide accurate attendance and leave tracking.
   To automate salary calculation and payroll generation.
   To improve HR workflow and reduce manual paperwork.
   To provide a secure login system for Admin and Employees.

3. Key Features
   Employee Management: Add, edit, and manage employee details.
   Attendance System: Track daily in/out time and working hours.
   Leave Management: Leave application, approval, and tracking.
   Payroll Processing: Automatic salary calculation with deductions.
   Performance Review: Evaluate employee performance and ratings.
   Dashboard: Role-based dashboard for Admin and Employees.
   Reports: View & download attendance and payroll reports.
   Notifications: Alerts for leave approvals, payroll updates, etc.

4. Technology Used
   Frontend: HTML5, CSS3, JavaScript, Bootstrap
   Backend: PHP (Core PHP or PHP OOP)
   Database: MySQL

5. Modules Overview
   Admin Login
   Employee Login
   Employee Management
   Attendance Management
   Leave Management
   Payroll Management
   Performance Evaluation
   Notifications

6. Conclusion
   This HR Management System reduces manual HR tasks and provides a complete digital solution for employee administration. It is efficient, scalable, and suitable for small to medium-sized organizations.
